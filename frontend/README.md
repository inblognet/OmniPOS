# OmniPOS - Offline First Point of Sale

## Tech Stack
- **React (Vite)**
- **Redux Toolkit** (State Management)
- **Dexie.js** (IndexedDB for Offline Storage)
- **Tailwind CSS** (Styling)

## Setup
1. Run `npm install`
2. Run `npm run dev`
3. Open `http://localhost:3000`

## Deployment (Vercel)
1. Build: `npm run build`
2. Output Directory: `dist`
3. Rewrites: `vercel.json` ensures SPA routing works.
